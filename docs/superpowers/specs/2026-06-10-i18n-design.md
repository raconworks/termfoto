# 中英文切换 — 设计文档

## 概述

添加中英文即时切换，`L` 键 toggle，自动检测系统 `$LANG` 作为默认语言。UI 文本集中管理于 `src/lang.rs`。

## 触发与切换

- **默认语言**：启动时读 `$LANG` 环境变量，以 `zh_` 开头 → 中文，否则英文
- **手动切换**：浏览器和全屏模式下按 `L` 键，即时 toggle 中/英
- 切换后无需清缓存或重新渲染——下一个 render tick 自然生效

## 架构：`src/lang.rs`

新增模块，零外部依赖。

```rust
#[derive(Debug, Clone, Copy, PartialEq)]
pub enum Lang { Zh, En }

impl Lang {
    pub fn detect() -> Self;
    pub fn toggle(&mut self);
    // 以下每个方法返回 &'static str
    pub fn browser_status_fmt(&self) -> &'static str;
    pub fn search_hint_matches(&self) -> &'static str;
    pub fn search_hint_empty(&self) -> &'static str;
    pub fn search_hint_none(&self) -> &'static str;
    pub fn preview_status_fmt(&self) -> &'static str;
    pub fn loading_text(&self) -> &'static str;
}
```

## 修改点

### `src/lang.rs` — 新建
- `Lang` 枚举 + 全部翻译方法 + `detect()` + `toggle()`
- 单元测试：detect 行为、toggle 翻转、所有翻译方法返回非空且中英文不同

### `src/app.rs`
- `App` struct 新增 `pub lang: Lang` 字段
- `new()` 参数加 `lang: Lang`，`handle_key()` 加 `L` 键分支调用 `self.lang.toggle()`

### `src/main.rs`
- `App::new()` 调用传入 `Lang::detect()`

### `src/ui/browser.rs`
- `BrowserView` 从 `app.lang` 取 `browser_status_fmt()` 构造状态栏

### `src/ui/search.rs`
- `SearchBar` 加 `lang: Lang` 字段，从 `lang` 取 `search_hint_*()` 方法

### `src/ui/preview.rs`
- `PreviewView` 从 `app.lang` 取 `preview_status_fmt()` 和 `loading_text()`

### `src/main.rs`
- 新增 `mod lang;` 声明，导入 `Lang`
